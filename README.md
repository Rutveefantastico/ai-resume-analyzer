# 🧠 AI Resume Analyzer

An AI-powered web application that analyzes resumes, calculates ATS score, and provides smart suggestions to improve job matching.

---

## 🚀 Features

- 📄 Resume Upload (PDF / Image)
- 🤖 AI-based Resume Analysis
- 📊 ATS Score Calculation
- 🧠 Skill Extraction
- 🎯 Job Role Matching
- 💡 AI Suggestions using Gemini API
- 📥 Download Report as PDF
- 📊 Dashboard Visualization
- ⚖️ Resume Comparison
- 🔐 User Authentication System

---

## 🛠 Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **AI:** Google Gemini API
- **Database:** SQLite
- **Libraries:** Pandas, NumPy, OpenCV, Pytesseract

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/ai-resume-analyzer.git
cd ai-resume-analyzer
pip install -r requirements.txt

---

## 🔑 Setup Environment Variables
Create a .env file:
GEMINI_API_KEY=your_api_key_here

---

## Run App
streamlit run app.py
